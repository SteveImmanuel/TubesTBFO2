/* Aplikasi CFG dan PDA pada Pengenalan Ekspresi Matematika */

/* Cara Menjalankan Program */
  User dapat run program secara langsung pada folder bin dan menjalankan program 'main.exe'

/* Format Input Pada Program */
  #) user dapat langsung memberi input ekspresi matematika yang ingin dikalkulasi
  #) Ekspresi matematika diinput tanpa ada spasi antar karakternya
  #) Input terdiri dari :
     - angka = {1,2,3,4,5,6,7,8,9,0}
     - simbol operasi aritmatika = {+,-,*,/}
     - simbol operasi perpangkatan = {^}
     - tanda negatif = {-}
     - tanda desimal = {.}
     - tanda kurung = {()}
     - tidak mengandung huruf, spasi, dan simbol-simbol lainnya.
  #) jika user selesai melakukan kalkulasi atau ingin keluar dari program, user dapat memberi input 'exit'

/* Keterangan Output Jika Perhitungan Gagal*/
  - SYNTAX ERROR
    -> ekspresi matematika yang diberikan terdapat huruf atau terdapat kesalahan dalam penggunaan operasi
       aritmatika sehingga ekspresi matematika tidak memungkinkan untuk dihitung

  - MATH ERROR
    -> ekspresi matematika yang diberikan tidak terdapat kesalahan dalam syntax, tetapi tidak dapat
       dihitung karena menghasilkan angka yang kompleks atau tidak mungkin dihitung
